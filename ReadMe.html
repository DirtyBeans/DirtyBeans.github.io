DBs6xColorKeyboard Instructions
  • Basic Rules
    • Device Intention:
      ( iPhone 4,4S,5 iPod 2 - 4, on iOS 6.0.0 higher ).
    • Must be Installed:
      ( Winterboard ), 
      ( ColorKeyboard "Purchased" ).
    • How to:
      ( With ColorKeyboad installed )
      Goto Colorkeyboard in settings
      Select Themes, select DBs6x
      Go back to ColorKeyboard main settings page, and select PopUp Key.
      Select either DBs6x-1 or 2, go back to main page.
      Make any other changes as neccessary. Once done press - (Save) - at top right. This will ask to Respring device, select Respring.
      Your Done
    • Incompatable Devices:
      ( iPhone 1stGen ),  ( iPhone 2 - 3GS ),( iPod 1stGen ),  ( iPad 1, 2 ).
