  • PKG.Backup
    (only for combined packages)

     If you use PKGbackup, your installed packages offered by DirtyBeans.com will not show as seperate installed packages.

  • Example :
    • Prior to using any of the DirtyBeans Pack(s), PKGBackup showed you had over 134 apps installed.
    • After install of any  DirtyBeans Pack(s), PKGBackup shows you only have 48 Apps installed
    • Whats Going On
      PKGBackup is only seeing the packages installed through Cydia, and not the actual installed list of apps. Therefore when you install a DBs pack , PKGBackup is only recording the installed DBs package and not the packages, mods, themes, etc, that were installed using the DBs pack. In other words. Lets say you installed 13 apps through cydia, then you installed any of the DBs packs which contained 10 of the apps you already installed. When you run PKGBackup you'll notice there is only 4 apps installed. 3 of which you already new and the new DBs package.

      *Solutions*
      1. There is no need to fear, all apps installed are still able to upgrade and delete individually.
      2. If you want to uninstall any of the apps. Do as you would normally do. (Individually or DBs package)
      3. If you uninstall any of the DBs packs. The apps that are associated with that pack will all be uninstalled as well.
      (Only on some packs. In general, be warned!!)
