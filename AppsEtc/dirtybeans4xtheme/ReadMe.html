DBs4xTheme Instructions
  • Basic Rules
    • Device Intention:
      ( iPhone 4 iOS 4.2.1 higher ).
    • Optional Devices
      ( iPhone 3G - 3GS iOS 4.0.1 higher ). "Some features not available"
    • Incompatable Devices:
      ( iPhone 1stGen ),  ( iPod 1stGen ).
    • Still Testing:
      ( iPod 2, 3, 4 ) ( iPad 1, 2 ).
  • How To
    • Required installs: WinterBoard
      ( You only need WinterBoard installed. ). DBs Package should not install without WinterBoard already installed.
      ( If you install any DBs package, Required packages like WinterBoard will automatically install )..............
    • Activation: Select desired Theme / Addon / Mod etc
      ( All DBs4x... are located in the WinterBoard directory once you install through cydia ). "On your device go to WinterBoard either through the icon or through settings, if you have it set up that way."
    • Selecting: WinterBoard / Select Themes
      ( WinterBoard functions in a top to bottom priority, which means whatever is on top will bypass what's below. )
      ( Example ) 2 themes installed , DBs4xTheme  and G.O.C. HD.  If DBs is above G.O.C. then G.O.C. will not be shown. And if both themes are selected then you may get conflicts and errors. Only select the Theme / Addon / Mod etc, you would like to see, and place them in the appropriate order.
    • After Selecting: Click the WinterBoard back button at the top
      ( WinterBoard should force a respring ) ( Make sure you respring to see desired Theme / Addon / Mod etc).
  • DBs4x_Theme.theme
    • ** Main theme **
    • Icon's:
      Over ( 155 ) DBs Made. (Apple and Cydia).
    • Remaining Apple Icon's:
      self-generating.
    • Remaining Cydia Icon's:
      non-generating.
    • Modified Elements:
      Calculator, Calendar, Clock, Compass, iPod, Mail, Maps, Search, Safari, SMS, Weather, and most viewable Thingamagigs.
    • Optional Mods:
      Install DBs4xWeather for live weather.
    • Optional Addons:
      Install DBs4xSBS. The DBs version of the SBSettings theme.
    • Compatible With:
      ( iPhone 4 ) ( iPhone 3G - 3GS ) ( iPod 2-4 ) ( iPad 1 ) ( iPad 2? ).
    • Change theme in WinterBoard / Select Themes.
  • DBs4x_Dated.Wallpaper.theme
    • Adds current date above your Dock.
    • Wallpaper automatically changes from 1 of so far ( 6 ), DBs custom made images.
    • Image(s) Cycle:
      every 36sec
    • Image Dimension: 640 x 960
    • To add your own wallpaper: Navigate to  " DBs4x_Dated.Wallpaper.theme / Wallpaper / "  folder using known methods. Replace any of the images in the folder. Make sure the image you replaced contains the original file name.
    • Compatible With:
      ( iPhone 4 ) ( iPhone 3G - 3GS ) ( iPod 2-4 ) ( iPad 1 ) ( iPad 2? ).
    • Change theme in WinterBoard / Select Themes.
  • DBs4x_DBs.Look.theme
    • Intended for the DBs staff.
    • Forced Icons, Logos, Wallpaper, Lockscreen, Etc.
    • Compatible With:
      ( iPhone 4 ).
    • Change theme in WinterBoard / Select Themes.
  • DBs4x_TTUnlock.theme
    • Tap to Un Lock
    • Lock Screen image:
      User, Change image the normal / standard way.
    • Compatible With:
      ( iPhone 4 ).
    • Semi Compatible With:
      ( iPhone 3G, 3GS ) ( iPod 1, 2, 3). No Tap, Must slide to unlock.
    • Change theme in WinterBoard / Select Themes.
