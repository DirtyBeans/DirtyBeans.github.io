    • Hack Sources
    • Hack Sources for new Install or Restore. Once you restore your iphone etc. Use this package to Install additional Sources.. This package will add an additional Cydia.list called hacksources.list,  as well as add several other sources to the private-etc-apt-sources.list.d folder.
      Intended for devices that are starting over, or you are new at jailbreaking and don't want to go through all the hassles..
    • Hack Sources Icons
    • After you install Hack Sources, some repo's want to force their icon packs onto you by making Cydia state you need an essential or complete upgrade. This is no fault of DirtyBeans. If you just install Hack Sources, you will notice this upgrade screen after the install. If you click Ignore, then go to Changes, you will notice the source(s) that you just installed, state there is an upgrade to their icon pack. Don't worry, just install Hack Sources Icons, to remove Cydia upgrade notice. This is not a Hack or Fix for this issue. Hack Sources Icons does the install of the icon pack as if you were to do this the normal way.
      Intended for devices that are starting over, or you are new at jailbreaking and don't want to go through all the hassles..
      * Denotes Icon packs
