    • Pre packed list of packages for new Install or Restore. Once you restore your iphone etc. Use this package to Install these additional packages.

    • Action Menu,
      Activator, Airplane SBSetting, AppBackup,
      Autocorrection SBSetting, Autolock SBSetting,

      pincrush, PreferenceLoader,
      SBSettings, SBSettingstoggles,
      Winterboard
    • Warning
      Tested on firmware 3.1.3 - 4.3.3
      (iPod, iPad, iPhone)
      Stable, as of   (June - 02 - 2011)

      *note*
      1. You should have a freshly restored iPhone / iPod / iPad with Cydia installed (of course), and have only added the DirtyBeans(repo) to Cydia's sources.
      2. Have installed the sources listed in the package Sources by DirtyBeans. If not install them prior, or just install DirtyBeans Sources package.
      3. Makes sure you are not going anywhere. That you have good WiFi, Power connected, and you do not close or allow the devices to shut down until the install is complete.
      This may take a min or two. If some packages do not install, just reinstall StarterPackage.
