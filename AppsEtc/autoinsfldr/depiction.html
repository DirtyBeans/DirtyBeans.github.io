Auto Install Folder
Install Cydia's autoinstall folder
    • This package will install the needed folders for Cydia to autoinstall Deb files.
    • 1. Copy deb package to:
        /private/var/root/Media/Cydia/AutoInstall/
    • °2. Rebooting device will install whatever packages are in that folder.
    • Warning:
    Some debs need the depend package as well.
