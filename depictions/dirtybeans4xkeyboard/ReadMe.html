DBs4xKeyboard Instructions
  • Basic Rules
    • Device Intention:
      ( iPhone 3G, 3GS, 4, iPod 2 - 4, iPad 1-2? on iOS 4.0.1 higher ).
    • Must be Installed:
      ( Winterboard ),  (  iAcces4China, for the device and firmware version ).
    • How to:
      ( With iAcces installed ). Goto Winterboard / Select Themes, select DBs4x_Keyboard,  Respring device, select sms to start the keyboard. click the globe to change keyboard views.

    • *There are a few ways to get this to work,  DirtyBeans has created a theme to go along with iAcces, and not iAcces !  For further info on how to use iAcces try Youtube.*
       
    • Incompatable Devices:
      ( iPhone 1stGen ),  ( iPod 1stGen ).
    • Still Testing:
      ( iPod 2, 3, 4 ) ( iPad 1, 2 ).
