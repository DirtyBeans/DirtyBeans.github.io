CSources2 ( Stable )

    • CSources2 is a Repo / Sources Manager, that allows full control over your sources.

    • Add as many sources as you want.
    • Manage sources for: Cydia, Installer, PostBox, Saily, Sileo, Zebra.
    • Disable/Enable or delete them.
    • Add notes.
    • Check if Source is active.
    • Package details.
    • Import cydia.list, zebra sources.list, ????.list file , Web Link.
    • Import Sileo.sources, ????.sources file , Web Link.
    • Add repo via Share Extension.
    • Backup Sources via iTunes , iCloud , Manually.
    • Sync between devices. (Device<->Device), (iCloud v933.1430-0) & up
    • View / Export Installed Packages. (v933.1451-0) & up
    • Restore Previously installed Packages. (v933.1451-0) & up
    • Auto List, Tweak Cydia features, & More !
